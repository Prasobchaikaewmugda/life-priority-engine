# LPE V1.12 Step 3 — Public User Guide V1

## Start

Open:

```text
https://life-priority-engine.streamlit.app
```

## Normal Daily Flow

1. ตั้งค่าชีวิต — ตรวจว่าครบ 5/5 โหมด
2. เช็คอินวันนี้ — ใส่บริบทวันนี้
3. แผนวันนี้ — ดูแผนและสิ่งที่ควรเลี่ยง
4. ทบทวนวันนี้ — บันทึกผลจริง
5. สำรองข้อมูล — export JSON

## If Data Is Empty

Go to สำรองข้อมูล and import your latest snapshot JSON.

Then return to ตั้งค่าชีวิต.

Acceptance:

```text
ครบ 5/5 โหมด — พร้อมใช้เป็นฐานชีวิต
```

## Backup Rule

After meaningful changes, export JSON.

Keep the latest JSON file somewhere safe.

## What This App Does Not Do

- No account
- No login
- No automatic cloud save
- No medical diagnosis
- No trading execution
